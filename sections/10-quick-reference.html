10 Rujukan Pantas (Cheat Sheet)

Satu halaman ringkas yang menggabungkan semua sintaks harian penting. Pastikan anda lihat label persekitaran PowerShell / WSL Ubuntu sebelum menyalin command.

📋 Kategori Command

  1. Kawalan Hos Windows & Sesi WSL
  2. Pengendalian Daemon & Servis Docker
  3. Pengurusan Harian Kontena
  4. Pembersihan Storan & Sisa Sistem

🖥️ Kawalan Hos Windows & Sesi WSL

PowerShell 
wsl --status

Semak status pemasangan & versi WSL — termasuk default version dan kernel version.

PowerShell 
wsl -d Ubuntu

Log masuk ke distro Ubuntu lalai. Gantikan Ubuntu dengan nama distro lain kalau ada.

PowerShell 
wsl --shutdown

Matikan semua sesi Linux serentak untuk bersihkan RAM. Sangat berguna bila nak jimat memori tanpa restart laptop.

🐳 Pengendalian Daemon & Servis Docker

WSL Ubuntu 
sudo service docker start

Aktifkan servis latar belakang Docker Engine. Wajib dijalankan selepas setiap restart laptop sebelum guna Docker.

WSL Ubuntu 
sudo service docker status

Semak sama ada Docker daemon sedang berjalan atau tidak. Output akan tunjuk "docker is running" atau "not running".

📦 Pengurusan Harian Kontena

WSL Ubuntu 
docker run -d -p 8080:80 --name web-node nginx

Luncurkan pelayan Nginx baharu di latar belakang, port 8080. -d = detached (latar belakang), -p 8080:80 = port 8080 komputer → port 80 container, --name web-node = nama container.

WSL Ubuntu 
docker ps

Lihat senarai container yang sedang AKTIF sahaja. Tak nampak container yang dah dihentikan.

WSL Ubuntu 
docker ps -a

Lihat SEMUA container — termasuk yang aktif dan yang telah dihentikan. Berguna untuk cari container lama yang terlupa.

🧹 Pembersihan Storan & Sisa Sistem

WSL Ubuntu 
docker stop web-node

Hentikan container secara selamat. Container masih boleh dihidupkan semula dengan docker start web-node.

WSL Ubuntu 
docker rm -f web-node

Padam container secara paksa. -f = force — padam walaupun container masih berjalan. Data dalam container akan hilang (kecuali disimpan dalam volume).

WSL Ubuntu 
docker system prune -a --volumes -f

Padam total semua cache, imej, volume, dan rangkaian yang tidak aktif. Sangat berkesan untuk bebaskan ruang disk NVMe. -a = semua imej tak guna, --volumes = padam juga volume terbiar, -f = force (tanpa confirm).

🔧 Troubleshooting

❌
Command docker tak jumpa?

Pastikan Docker daemon sedang berjalan: sudo service docker start. Kalau masih tak jumpa, mungkin Docker belum dipasang — rujuk Section 05 — Docker Setup.

❌
Permission denied walaupun dah guna sudo?

Kalau anda perlu guna sudo untuk setiap command docker, ini maknanya user anda belum ditambah ke group docker. Rujuk Troubleshooting penuh untuk penyelesaian.
